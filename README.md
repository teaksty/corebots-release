# CoreBots

Лаунчер с ботами для GTA 5 RP. 10 ботов в одном приложении: качайся, работай
и фармь, пока занимаешься своими делами.

## Скачать

**[Скачать последнюю версию](https://github.com/teaksty/corebots-release/releases/latest)**

Файл `CoreBots-Setup.exe` — обычный установщик, ставится за пару кликов,
права администратора не нужны.

## Что внутри

Качалка · Стройка · Порт · Шахта · Колесо · Ферма · Кулинария · Деморган ·
Анти АФК · Авто КПК

Плюс встроенные разделы: калькулятор перекупа, таймеры, бонус-поинты и
настройки внешнего вида.

## Требования

- Windows 10 или 11, 64-бит
- Ключ активации — выдаётся при покупке на [corebots.org](https://corebots.org)

## Как установить

1. Скачать `CoreBots-Setup.exe` из релизов.
2. Запустить. Если Windows покажет «Неизвестный издатель» — «Подробнее» →
   «Выполнить в любом случае» (приложение пока без сертификата подписи).
3. При первом запуске ввести ключ активации.

Все данные — сохранения разделов, настройки, лицензия — хранятся локально в
папке приложения. Никуда не отправляются.

## Поддержка

Вопросы и ключи — на сайте [corebots.org](https://corebots.org).
